Bem-vindo ao Módulo 4: Laços de Repetição (for, while)

Vamos aprender sobre os laços de repetição em JavaScript, como os laços for, while e do-while!

Introdução aos Laços de Repetição

Laços de repetição são estruturas de controle essenciais na programação. Eles permitem que você execute um bloco de código várias vezes de maneira eficiente. No JavaScript, os principais laços de repetição são for, while e do-while. Cada um desses laços tem características específicas que os tornam úteis em diferentes cenários.

1. O Laço `for`

O laço for é usado quando sabemos o número exato de vezes que queremos executar um bloco de código. Sua sintaxe é a seguinte:


                for (inicialização; condição; incremento) {
                    // Bloco de código a ser repetido
                }

Exemplo básico de um laço for:


                for (let i = 0; i < 5; i++) {
                    console.log(i);
                }

Este código vai imprimir os números de 0 a 4, pois a condição i < 5 é verdadeira enquanto i for menor que 5.

2. O Laço `while`

O laço while é usado quando não sabemos exatamente o número de vezes que o código será executado, mas queremos que o código continue enquanto uma condição for verdadeira.


                while (condição) {
                    // Bloco de código a ser repetido
                }

Exemplo de uso do laço while:


                let i = 0;
                while (i < 5) {
                    console.log(i);
                    i++;
                }

Neste exemplo, o código vai imprimir os números de 0 a 4, pois a condição i < 5 será verificada antes de cada iteração.

3. O Laço `do-while`

O laço do-while é semelhante ao while, mas a condição é verificada após a execução do bloco de código. Isso significa que o código dentro do laço será executado pelo menos uma vez.


                do {
                    // Bloco de código a ser repetido
                } while (condição);

Exemplo de uso do laço do-while:


                let i = 0;
                do {
                    console.log(i);
                    i++;
                } while (i < 5);

Assim como o while, este código também imprime os números de 0 a 4, mas garante que o bloco de código seja executado pelo menos uma vez, mesmo que a condição já seja falsa.

4. Controle de Fluxo dentro dos Laços

Dentro de um laço, podemos usar as palavras-chave break e continue para controlar o fluxo de execução.

  • break: Interrompe a execução do laço imediatamente.
  • continue: Pula a iteração atual e continua com a próxima.

                for (let i = 0; i < 10; i++) {
                    if (i === 5) {
                        break; // Interrompe o laço quando i for igual a 5
                    }
                    console.log(i);
                }

O código acima vai imprimir os números de 0 a 4, pois o laço é interrompido quando i é igual a 5.


                for (let i = 0; i < 10; i++) {
                    if (i === 5) {
                        continue; // Pula a iteração quando i for igual a 5
                    }
                    console.log(i);
                }

Este código vai imprimir os números de 0 a 9, exceto 5, pois a iteração é pulada quando i === 5.

5. Exemplos Práticos de Laços

Abaixo estão alguns exemplos práticos do uso de laços para resolver problemas do cotidiano:


                // Somar os números de 1 a 10
                let soma = 0;
                for (let i = 1; i <= 10; i++) {
                    soma += i;
                }
                console.log(soma);  // Resultado: 55
            

                // Encontrar números pares entre 1 e 20
                for (let i = 1; i <= 20; i++) {
                    if (i % 2 === 0) {
                        console.log(i);  // Imprime números pares
                    }
                }
