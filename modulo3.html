Bem-vindo ao Módulo 3: Estruturas Condicionais

Vamos aprender sobre as estruturas condicionais em JavaScript!

O que são Estruturas Condicionais?

Estruturas condicionais são blocos de código que executam diferentes ações dependendo de uma condição verdadeira ou falsa. Elas são fundamentais para a lógica de programação e permitem que o fluxo de execução do código seja alterado com base em decisões.

1. A Estrutura `if`

A estrutura `if` permite que você execute um bloco de código se uma condição for verdadeira. O código dentro do `if` será executado apenas quando a condição definida for verdadeira.


                let idade = 18;
                if (idade >= 18) {
                    console.log("Você é maior de idade.");
                }

No exemplo acima, se a variável `idade` for maior ou igual a 18, o programa exibirá "Você é maior de idade". Se a condição for falsa, o código dentro do `if` não será executado.

2. A Estrutura `else`

O `else` permite que você defina um bloco de código alternativo, que será executado caso a condição do `if` seja falsa.


                let idade = 16;
                if (idade >= 18) {
                    console.log("Você é maior de idade.");
                } else {
                    console.log("Você é menor de idade.");
                }

Se a condição do `if` for falsa, o código dentro do `else` será executado.

3. Estruturas `else if`

Você pode usar o `else if` para testar várias condições diferentes. Caso a condição do `if` seja falsa, o código verifica as condições do `else if` e, se alguma delas for verdadeira, o bloco correspondente será executado.


                let nota = 75;
                if (nota >= 90) {
                    console.log("Nota A");
                } else if (nota >= 70) {
                    console.log("Nota B");
                } else if (nota >= 50) {
                    console.log("Nota C");
                } else {
                    console.log("Nota D");
                }

No exemplo acima, o programa verifica as condições em sequência até encontrar uma que seja verdadeira.

4. Operadores Lógicos

Os operadores lógicos permitem combinar múltiplas condições. Os mais comuns são:

  • AND (&&): Retorna verdadeiro se ambas as condições forem verdadeiras.
  • OR (||): Retorna verdadeiro se pelo menos uma das condições for verdadeira.
  • NOT (!): Inverte o valor lógico da condição.

                let idade = 20;
                let temCarro = true;
                if (idade >= 18 && temCarro) {
                    console.log("Você pode dirigir.");
                } else {
                    console.log("Você não pode dirigir.");
                }

Esse exemplo usa o operador `&&` para verificar se ambas as condições são verdadeiras.

5. A Estrutura `switch`

A estrutura `switch` é uma forma de simplificar múltiplos testes condicionais. Ela verifica uma variável contra várias opções e executa o código correspondente à primeira condição que for verdadeira.


                let dia = "domingo";
                switch (dia) {
                    case "segunda-feira":
                        console.log("Hoje é segunda-feira.");
                        break;
                    case "domingo":
                        console.log("Hoje é domingo.");
                        break;
                    default:
                        console.log("Não é um dia da semana.");
                }

O `switch` é útil quando você tem várias opções e deseja comparar uma variável com diferentes valores possíveis.

6. Exemplos Práticos com `if`, `else`, `switch`

Agora que você já conhece as estruturas condicionais, vamos ver alguns exemplos práticos de como usá-las:


                // Exemplo 1: Verificando a idade
                let idade = 21;
                if (idade < 18) {
                    console.log("Você é menor de idade.");
                } else {
                    console.log("Você é maior de idade.");
                }

                // Exemplo 2: Definindo a classificação de uma nota
                let nota = 85;
                switch (true) {
                    case (nota >= 90):
                        console.log("Nota A");
                        break;
                    case (nota >= 70):
                        console.log("Nota B");
                        break;
                    case (nota >= 50):
                        console.log("Nota C");
                        break;
                    default:
                        console.log("Nota D");
                }
