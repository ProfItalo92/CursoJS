Bem-vindo ao Módulo 7: Manipulação de DOM

Vamos aprender sobre como manipular o DOM (Document Object Model) com JavaScript!

Introdução ao DOM

O DOM (Document Object Model) é a interface que permite a interação entre o JavaScript e a estrutura HTML de uma página. Com o DOM, podemos acessar, modificar, adicionar ou excluir elementos da página, além de responder a interações do usuário, como cliques e digitação.

1. O que é o DOM?

O DOM é uma representação em árvore da estrutura de um documento HTML. Cada elemento, atributo e texto no HTML é representado como um nó no DOM, e podemos usar JavaScript para manipular esses nós.


                // Exemplo básico de acesso ao DOM
                let titulo = document.getElementById('titulo');
                console.log(titulo.innerText);  // Exibe o conteúdo do título

2. Como Funciona o DOM em JavaScript

Quando você carrega uma página web no navegador, o navegador cria uma árvore DOM, que é uma estrutura de dados que representa todos os elementos HTML da página. Com JavaScript, você pode acessar e manipular essa árvore para alterar a aparência e o comportamento da página.

3. Selecionando Elementos do DOM

Para manipular os elementos no DOM, você precisa selecioná-los primeiro. Existem várias maneiras de selecionar elementos:

  • document.getElementById(id): Seleciona um único elemento pelo seu ID.
  • document.getElementsByClassName(class): Seleciona todos os elementos que possuem a classe especificada.
  • document.querySelector(selector): Seleciona o primeiro elemento que corresponde ao seletor CSS fornecido.
  • document.querySelectorAll(selector): Seleciona todos os elementos que correspondem ao seletor CSS fornecido.

                // Selecionando um elemento pelo ID
                let titulo = document.getElementById('titulo');
                console.log(titulo);

                // Selecionando elementos por classe
                let paragrafos = document.getElementsByClassName('paragrafo');
                console.log(paragrafos);

4. Manipulando Conteúdo e Atributos

Uma vez que você tenha selecionado um elemento, pode modificar seu conteúdo ou atributos. O conteúdo pode ser acessado ou modificado através de propriedades como innerHTML, innerText ou value (para campos de formulário).


                // Modificando o conteúdo de um elemento
                let titulo = document.getElementById('titulo');
                titulo.innerText = "Novo Título";

                // Modificando o valor de um campo de entrada
                let input = document.getElementById('nome');
                input.value = "Novo Nome";

5. Eventos no DOM

Os eventos são ações que ocorrem na página, como cliques, digitação ou rolagem, e podemos configurar manipuladores de eventos para executar código quando esses eventos ocorrem.


                // Adicionando um evento de clique
                let botao = document.getElementById('meuBotao');
                botao.addEventListener('click', function() {
                    alert("Botão clicado!");
                });

6. Alterando Estilos com JavaScript

Você pode alterar o estilo de um elemento HTML diretamente através do DOM, usando a propriedade style.


                // Alterando o estilo de um elemento
                let titulo = document.getElementById('titulo');
                titulo.style.color = "red";
                titulo.style.fontSize = "24px";

7. Criando e Removendo Elementos Dinamicamente

Você pode adicionar novos elementos à página ou remover elementos existentes com JavaScript. Para criar um novo elemento, use o método createElement, e para adicionar o elemento à página, use appendChild.


                // Criando um novo parágrafo e adicionando à página
                let paragrafo = document.createElement('p');
                paragrafo.innerText = "Este é um novo parágrafo!";
                document.body.appendChild(paragrafo);

Para remover um elemento, você pode usar o método removeChild:


                let paragrafo = document.getElementById('paragrafo');
                document.body.removeChild(paragrafo);

8. Exemplos Práticos de Manipulação do DOM

Aqui estão alguns exemplos práticos para você testar:


                // Alterar o título da página com um clique
                let botao = document.getElementById('meuBotao');
                botao.addEventListener('click', function() {
                    let titulo = document.getElementById('titulo');
                    titulo.innerText = "Você clicou no botão!";
                });
