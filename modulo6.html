Bem-vindo ao Módulo 6: Objetos e Arrays

Vamos aprender sobre objetos e arrays em JavaScript!

Introdução aos Objetos e Arrays

Objetos e arrays são dois dos tipos de dados mais importantes e usados no JavaScript. Eles permitem que você armazene e manipule dados de forma eficiente e flexível. Neste módulo, vamos aprender como usar objetos e arrays, suas diferenças, como acessá-los e manipulá-los, e como usá-los para resolver problemas do mundo real.

1. Objetos em JavaScript

Objetos são coleções de pares chave-valor. Em JavaScript, você pode definir um objeto utilizando chaves {} e dentro delas, você coloca as propriedades e seus respectivos valores.


                let pessoa = {
                    nome: "João",
                    idade: 30,
                    profissao: "Programador"
                };
                console.log(pessoa.nome);  // Resultado: João

O exemplo acima define um objeto chamado pessoa com três propriedades: nome, idade e profissao. Para acessar uma propriedade do objeto, usamos a notação de ponto.

Acessando Propriedades de Objetos


                let pessoa = {
                    nome: "Maria",
                    idade: 25,
                    cidade: "São Paulo"
                };
                console.log(pessoa.idade);  // Resultado: 25
                console.log(pessoa["cidade"]);  // Resultado: São Paulo

Além da notação de ponto, podemos acessar propriedades utilizando colchetes, o que é útil quando as propriedades têm nomes com espaços ou caracteres especiais.

Métodos de Objetos

Objetos também podem conter métodos, ou seja, funções que pertencem a um objeto.


                let carro = {
                    marca: "Toyota",
                    modelo: "Corolla",
                    mostrarInfo: function() {
                        console.log(this.marca + " " + this.modelo);
                    }
                };
                carro.mostrarInfo();  // Resultado: Toyota Corolla

Objetos Aninhados

Objetos podem conter outros objetos dentro deles. Isso permite criar estruturas de dados complexas.


                let aluno = {
                    nome: "Pedro",
                    idade: 20,
                    endereco: {
                        rua: "Rua X",
                        numero: 123
                    }
                };
                console.log(aluno.endereco.rua);  // Resultado: Rua X

2. Arrays em JavaScript

Arrays são listas ordenadas de valores. Em JavaScript, você pode criar um array usando colchetes [].


                let numeros = [1, 2, 3, 4, 5];
                console.log(numeros[0]);  // Resultado: 1

O array acima contém cinco números. A indexação começa em 0, ou seja, o primeiro elemento do array é acessado com o índice 0.

Acessando Elementos de Arrays


                let frutas = ["maçã", "banana", "laranja"];
                console.log(frutas[1]);  // Resultado: banana

Métodos de Arrays

Arrays têm muitos métodos úteis para manipulação, como push() para adicionar elementos ao final e pop() para remover o último elemento.


                let numeros = [1, 2, 3];
                numeros.push(4);  // Adiciona 4 ao final
                console.log(numeros);  // Resultado: [1, 2, 3, 4]
                numeros.pop();  // Remove o último elemento
                console.log(numeros);  // Resultado: [1, 2, 3]

Arrays Aninhados

Arrays também podem conter outros arrays dentro deles, criando uma estrutura de dados multidimensional.


                let matriz = [
                    [1, 2, 3],
                    [4, 5, 6],
                    [7, 8, 9]
                ];
                console.log(matriz[1][2]);  // Resultado: 6

Iterando sobre Arrays

Você pode usar loops para iterar sobre os elementos de um array. O laço for é muito útil para isso.


                let frutas = ["maçã", "banana", "laranja"];
                for (let i = 0; i < frutas.length; i++) {
                    console.log(frutas[i]);
                }

3. Diferenças entre Objetos e Arrays

Embora tanto objetos quanto arrays sejam usados para armazenar dados, eles têm diferenças significativas:

  • Objetos: São usados para armazenar dados de forma estruturada, usando pares chave-valor.
  • Arrays: São usados para armazenar listas ordenadas de dados.
  • Objetos: Permitem acessar valores através de chaves, que podem ser strings ou números.
  • Arrays: Permitem acessar valores através de índices numéricos.

4. Exemplos Práticos

Aqui estão alguns exemplos mais complexos de como usar objetos e arrays:


                let aluno = {
                    nome: "Maria",
                    idade: 22,
                    cursos: ["Matemática", "Física"],
                    mostrarCursos: function() {
                        console.log(this.cursos.join(", "));
                    }
                };
                aluno.mostrarCursos();  // Resultado: Matemática, Física
            

                let equipe = [
                    { nome: "João", cargo: "Desenvolvedor" },
                    { nome: "Ana", cargo: "Designer" },
                    { nome: "Carlos", cargo: "Gerente" }
                ];
                for (let i = 0; i < equipe.length; i++) {
                    console.log(equipe[i].nome + " é " + equipe[i].cargo);
                }
