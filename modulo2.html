Introdução às Variáveis e Tipos de Dados

O conceito de variáveis e tipos de dados são fundamentais para qualquer linguagem de programação, e JavaScript não é diferente. As variáveis são "caixas" onde você pode armazenar valores, e os tipos de dados definem que tipo de valor pode ser armazenado. No JavaScript, você pode trabalhar com diversos tipos de dados, como números, textos, booleanos, arrays e objetos. Este módulo vai cobrir tudo isso com exemplos e explicações detalhadas.

1. O que são Variáveis?

Variáveis são fundamentais em programação. Elas permitem que você armazene valores, como números e textos, para usá-los posteriormente no seu código. Em JavaScript, você pode declarar variáveis de três formas:

  • var: Uma maneira antiga de declarar variáveis, mas com escopo global ou de função.
  • let: Usado para declarar variáveis com escopo de bloco.
  • const: Usado para declarar variáveis cujo valor não pode ser alterado.

                let idade = 30;  // Exemplo com let
                const nome = "Maria";  // Exemplo com const
                var ativo = true;  // Exemplo com var

2. Tipos de Dados Primitivos

Os tipos de dados primitivos são os mais simples no JavaScript e não podem ser alterados diretamente. São usados para armazenar valores simples, como números, texto e valores lógicos.

  • Number: Usado para números, tanto inteiros quanto decimais. Exemplo: let x = 5.25;
  • String: Usado para armazenar texto. Exemplo: let nome = "João";
  • Boolean: Usado para armazenar valores lógicos: true ou false.
  • Null: Representa a ausência de valor ou objeto.
  • Undefined: Usado para variáveis que foram declaradas, mas não inicializadas.
  • Symbol: Tipo de dado único e imutável, usado principalmente para identificar propriedades de objetos.

Exemplo de Tipos de Dados Primitivos


                let numero = 42;           // Tipo Number
                let texto = "Olá, Mundo";  // Tipo String
                let ativo = true;          // Tipo Boolean
                let vazio = null;          // Tipo Null
                let indefinido;            // Tipo Undefined (não atribuído)

3. Conversão de Tipos de Dados

Às vezes, é necessário converter um tipo de dado para outro. O JavaScript oferece funções como Number(), String(), e Boolean() para realizar essas conversões.


                let numeroTexto = "123";
                let numeroConvertido = Number(numeroTexto);  // Converte para Number
                let booleano = Boolean("");  // Converte uma string vazia para false

4. Arrays e Objetos

Arrays e objetos são tipos de dados compostos em JavaScript. Eles podem armazenar múltiplos valores em uma única variável.

  • Arrays: Listas ordenadas de valores, acessados por um índice numérico.
  • Objetos: Coleções de propriedades e valores, onde as propriedades são identificadas por chaves (strings).

                let numeros = [1, 2, 3, 4]; // Exemplo de Array
                let pessoa = { nome: "João", idade: 25 }; // Exemplo de Objeto

5. Trabalhando com Variáveis

Em JavaScript, as variáveis podem ser manipuladas de diversas maneiras. A seguir estão exemplos de operações que podemos fazer com variáveis:


                let x = 10;
                let y = 5;
                let soma = x + y;  // Soma
                let produto = x * y;  // Multiplicação

6. Escopo das Variáveis

O escopo de uma variável define onde ela pode ser acessada dentro do código. As variáveis podem ter escopo global ou local, dependendo de onde são declaradas.

  • Escopo Global: A variável pode ser acessada em qualquer parte do código.
  • Escopo Local: A variável só pode ser acessada dentro da função ou bloco onde foi declarada.
